## Template-specific features

### 1. Best Tools Listicle ✅
- Tool cards with rank, G2 rating, pros/cons, pricing
- Category jump pills (horizontal scroll nav)
- TL;DR comparison table at top
- Methodology section with scoring criteria
- Featured card = Overloop only

### 2. Alternatives
- Hero: "[Tool] is great, but not for everyone" positioning
- Quick comparison table (us vs them vs alternatives)
- Each alternative as a card: logo, pricing, pros/cons, verdict
- "Why teams switch from [Tool]" section
- Migration checklist
- Every card links to /versus/ page if exists

### 3. Pillar / Ultimate Guide
- Chapter navigation (numbered, progress tracking)
- Longer sections with sub-TOC per chapter
- Key takeaways box per section
- Downloadable checklist/PDF CTA
- Expert quotes scattered throughout
- Internal links to related articles per section

### 4. How-To Tutorial
- Numbered steps with visual step indicators (gradient circles)
- Prerequisite checklist at top
- Time-to-complete estimate
- Screenshot placeholders per step
- "What you'll learn" summary box
- Video embed placeholder

### 5. X vs Y (V2)
- Current V1 template + add: expert callouts, tested-by banner
- Feature comparison table (check/cross/partial)
- Pricing section with calculator link
- "When to choose A vs B" decision matrix
- Migration section
- Verdict with CTA

### 6. Templates & Swipe Files
- Copyable template cards (one-click copy button)
- Category tabs (email, LinkedIn, follow-up, etc.)
- Each template: subject line + body + variables highlighted
- "Personalize this" tips per template
- Performance stats (open rate, reply rate) if available
- Download all as PDF CTA

### 7. Data List / Directory
- Searchable/filterable table
- Categories/tags for filtering
- Each entry: name, description, use case
- "table-keep" for mobile (no card collapse)
- Export/download CTA
- Related lists section

### 8. Statistics & Benchmarks
- Key stats as hero cards (big numbers, gradient text)
- Source citations on every stat
- Quotable stat cards (shareable, with brand watermark)
- Interactive chart placeholders
- Methodology section
- Year-over-year comparison

### 9. Best Agency Roundup
- Similar to Best Tools but for agencies/services
- Agency cards: logo, specialties, pricing range, location
- "Best for" tags per agency
- Comparison table (features × agencies)
- Overloop as the self-serve alternative

### 10. Strategy / Playbook
- Framework visualization (steps, funnel, flowchart)
- Action items per section (checkboxes)
- Resource links per step
- "Run this in Overloop" CTA per section
- Downloadable playbook PDF CTA

### 11. Lead Gen for X Business
- Industry-specific hero (vertical-aware)
- ICP definition section
- Channel recommendations
- Templates section (email + LinkedIn)
- Case study from that industry
- "Start with Overloop" CTA

### 12. What is X? (Definition)
- Clean definition box at top (AEO-critical, quotable)
- "In this guide" TOC
- Sections: What, Why, How, Examples, FAQ
- Related definitions sidebar
- Expert quote on the topic

### 13. Role-Based Guide
- Role-specific hero ("The SDR's Guide to...")
- Day-in-the-life workflow section
- Tools recommended per task
- KPIs/metrics section
- Career path / growth section

### 14. Tactical Articles
- Short-form, focused, actionable
- Key takeaway box at top
- 3-7 clear points/tips
- Each point: heading + explanation + action item
- Minimal — no heavy components, fast to read

### 15. Tool Review (Single)
- Verdict box at top (score, one-line, pros/cons summary)
- Scoring table (features, ease of use, pricing, support, etc.)
- Detailed sections per criterion
- Pricing breakdown table
- "Overloop as alternative" section with comparison
- User reviews section (G2/Capterra quotes)

## Design decisions

- Featured card = Overloop only
- Pro tips = named expert with photo (rotate between 3)
- TL;DR card = removed (replaced by author intro)
- Update log = automated via cron agent (monthly pricing checks)
- Palette = indigo #6366F1 → purple #A855F7 gradient
- Fonts = Epilogue (sans) + Playfair Display italic (accent) + JetBrains Mono (stats/code)
